Tech Stack

Vue 3 + Vite
Composition API, script setup, reactive refs & computed properties
Axios
HTTP client, centralized in api.js, base URL http://localhost:8001/api
FastAPI + Uvicorn
Python ASGI server, auto-generated OpenAPI docs at /docs
Pydantic v2
Request/response validation, typed models for all data entities
JSON Mock Data
7 files loaded by mock_data.py at startup, filtered in-memory
pytest + FastAPI TestClient
Backend unit tests in tests/backend/
i18n (en / ja)
useI18n composable, locale persisted in localStorage

Vue Composables (Shared State)

useFilters.js
Global filter state: selectedPeriod, selectedLocation, selectedCategory, selectedStatus. Provides getCurrentFilters() consumed by all views to build API requests.
useAuth.js
Mock user session: currentUser, isAuthenticated, getInitials(). Supplies profile data to ProfileMenu and TasksModal.
useI18n.js
Internationalization: t(key), setLocale(), translateProductName(), translateWarehouse(). Switches between English and Japanese, persists in localStorage.

The 4 Global Filters

Filter Frontend State API Param Values Applies To
Time Period selectedPeriod month all, 2025-01 … 2025-12, Q1-2025 … Q4-2025 Orders, Dashboard
Warehouse selectedLocation warehouse all, San Francisco, London, Tokyo Inventory, Orders, Dashboard
Category selectedCategory category all, circuit boards, sensors, actuators, controllers, power supplies Inventory, Orders, Dashboard
Order Status selectedStatus status all, delivered, shipped, processing, backordered Orders, Dashboard
